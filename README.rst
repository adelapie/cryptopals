Cryptopals
==========

Solutions to the `cryptopals crypto challenges`_.  You may want to
look at the notes_.  Some files produce interesting text when run,
others just test assertions.  A few have to be run in parallel,
sometimes with environment variables set.

.. _cryptopals crypto challenges: https://www.cryptopals.com/
.. _notes: https://github.com/wasamasa/cryptopals/blob/master/notes.md
